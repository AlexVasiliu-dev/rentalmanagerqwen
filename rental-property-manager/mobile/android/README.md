# Android App Configuration

This folder contains Android-specific native code and configuration files for the RentManager Android app.

## Building for Android

### Requirements
- Java Development Kit (JDK) 11+
- Android Studio with Android SDK
- Android 6.0 (API level 23)+ deployment target

### Development Build

```bash
# Using Expo
npx expo run:android

# Or using EAS Build
eas build --platform android --profile development
```

### Production Build

```bash
# Using EAS Build
eas build --platform android --profile production
```

## Configuration Files

- `AndroidManifest.xml` - Android app manifest (generated by Expo)
- `build.gradle` - Gradle build configuration (generated by Expo)
- `*.gradle` - Gradle files (generated by Expo)

## Google Play Store Submission

1. Build the app using EAS Build (produces .aab file)
2. Upload to Google Play Console
3. Submit for review

## Icons and Splash Screen

Place your Android app icons in the `assets/` folder:
- `icon.png` - App icon (512x512)
- `adaptive-icon.png` - Adaptive icon (1024x1024)
- `splash.png` - Splash screen (2048x2048)

## APK vs AAB

- **APK**: For testing and sideloading
- **AAB**: Required for Google Play Store submission

Generate APK for testing:
```bash
eas build --platform android --profile preview
```
