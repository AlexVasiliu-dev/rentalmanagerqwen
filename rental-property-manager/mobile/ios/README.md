# iOS App Configuration

This folder contains iOS-specific native code and configuration files for the RentManager iOS app.

## Building for iOS

### Requirements
- macOS with Xcode installed
- Apple Developer account
- iOS 13.0+ deployment target

### Development Build

```bash
# Using Expo
npx expo run:ios

# Or using EAS Build
eas build --platform ios --profile development
```

### Production Build

```bash
# Using EAS Build
eas build --platform ios --profile production
```

## Configuration Files

- `Info.plist` - iOS app info (generated by Expo)
- `Podfile` - CocoaPods dependencies (generated by Expo)
- `*.xcodeproj` - Xcode project (generated by Expo)

## App Store Submission

1. Build the app using EAS Build
2. Upload to App Store Connect
3. Submit for review

## Icons and Splash Screen

Place your iOS app icons in the `assets/` folder:
- `icon.png` - App icon (1024x1024)
- `adaptive-icon.png` - Adaptive icon (1024x1024)
- `splash.png` - Splash screen (2048x2048)
